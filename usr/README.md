# Socio-Economic Development of Tribal Communities

A cross-platform flutter application serving as a comprehensive portal and dashboard for monitoring and managing the socio-economic development of tribal communities.

## Features
- **Key Indicators Dashboard**: Overview of socio-economic progress, literacy rates, employment, and healthcare metrics.
- **Communities Directory**: Detailed profiles for different tribal communities, including their specific demographic and socio-economic statistics.
- **Projects & Reports Tracker**: Track the status, budget, and impact of various development initiatives and download comprehensive project reports.
- **Responsive Layout**: Designed to work seamlessly across mobile devices, tablets, and desktop environments.

## Tech Stack
- Flutter (Dart)
- Material 3 Design
- Cross-platform support (Web, iOS, Android, Desktop)

## Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Launch the app using `flutter run`

## Development Notes
The app currently uses mock data for rapid prototyping and demonstration purposes. In a production environment, this should be connected to a robust backend or headless CMS managing the real-time socio-economic indicator data.
